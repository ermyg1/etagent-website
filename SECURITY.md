# Security Policy

## Reporting a Vulnerability

Please report suspected security vulnerabilities privately to the project
maintainers at security@etagent.io. This email address will become active once
the production domain email system is configured. Do not open a public issue for
security-sensitive findings.

Include as much relevant detail as possible:

- A summary of the issue.
- Steps to reproduce or verify the behavior.
- Potential impact.
- Affected files, routes, dependencies, or configurations.

Maintainers will review reports, assess impact, and coordinate remediation
before public disclosure when appropriate.
